## Instalação

Para iniciar o projeto, após feito o [download](https://github.com/VitorSMaia/Calculator) dos arquivos, acesse a pasta raiz e execulte os seguinte comandos.

```sh
dotnet restore
```
Em seguida:
```sh
dotnet build
```
Para execultar o projeto:
```sh
dotnet run
```
